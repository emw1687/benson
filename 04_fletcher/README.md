### Finding new friends in Seattle

#### Tools
* [nltk](http://www.nltk.org/)
* [Pandas](http://pandas.pydata.org/)
* [Seaborn](http://seaborn.pydata.org/index.html)
* [Scikit-learn](http://scikit-learn.org/stable/)
* [Gensim](https://radimrehurek.com/gensim/)

#### Data Source
[Seattle Airbnb Open Data](https://www.kaggle.com/airbnb/seattle)

#### Pipeline
1. [Remove stopwords and lemmatize host descriptions from Seattle AirBnB data](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/01_preprocessing.ipynb)
2. [Cluster host descriptions using kmeans](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/02_kmeans.ipynb)
3. [Complete topic modeling using NMF and LDA](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/03_topic_modeling.ipynb)
4. [Cluster host descriptions using DBSCAN ](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/05_dbscan.ipynb)
5. [Cluster host descriptions using hierarchical clustering](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/06_hierarchical.ipynb)
6. [Create MVP for recommendation system with spectrum of similarity](https://github.com/emw1687/metis_projects/blob/master/04_fletcher/notebooks/04_nmf_demo.ipynb)
